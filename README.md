# ShopEZ - Backend E-Commerce 
### Versión 2.0

**ShopEZ** es un sistema backend modular para comercio electrónico desarrollado en **Go (Golang)**.

## 🚀 Características
* **Gestión de Usuarios:** Control de perfiles y roles de administrador y cliente.
* **Control de Inventario:** Administración de productos y stock de la tienda.
* **Descuentos Dinámicos:** Aplicación de lógica flexible mediante interfaces (incluyendo descuento del 10% UIDE).

## 🛠️ Ejecución
Para correr el proyecto localmente, ejecuta en tu terminal:

```bash
go run main.go
